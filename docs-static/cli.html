CLI Usage

Use the CLI to explain errors, predict problems, and track history.

Explain

dbug explain "NameError: name 'x' is not defined"
dbug explain -f path/to/error.log

Predict

dbug predict path/to/file.py --severity high --limit 5

History

dbug history

dbug history --stats

dbug history --search NameError

Train

dbug train --interactive
dbug train --ml

Config

dbug config --show

dbug config ai_provider grok

dbug config grok_api_key YOUR_KEY
