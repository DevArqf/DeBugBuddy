Privacy and Security

DeBugBuddy is local-first and only sends data when you opt into AI.

Local by default

  • Parsing, pattern matching, prediction, and history stay on your machine.
  • AI is optional and controlled by your configuration.

History storage

  • Windows: %USERPROFILE%\.debugbuddy\history.db
  • macOS/Linux: ~/.debugbuddy/history.db

Safe usage

  • Avoid sending secrets to AI providers.
  • Rotate API keys if compromised.
Report vulnerabilities via GitHub Security Advisories.
