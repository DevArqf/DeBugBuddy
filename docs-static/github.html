GitHub Integration

Search issues precisely or report an error to a repo.

Accurate search

dbug github search "TypeError: unsupported operand type(s) for +" \
  -l python --repo yourorg/yourrepo --exact --include-closed
  • Provide the full error type and a short message.
  • Scope to a repo for the best precision.
  • Use --exact for strict matching.

Report an issue

dbug github report "Your error text" -r owner/name
dbug config github_token YOUR_TOKEN
GitHub search prioritizes exact matches and repo scope.
